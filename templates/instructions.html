Generate a Gemini API Key

1. Open Google AI Studio — API Keys.
2. Sign in with your Google account, if prompted.
3. Click "Create API key" and copy your key.
4. Return to the chat page and paste your key into the input field. The key is stored in your browser (localStorage) and will be used for your requests.
5. If you prefer server-wide configuration, set the environment variable GOOGLE_GENAI_API_KEY on the server.
